# Animated 404 Error Page

A creative 404 error page built using HTML and CSS featuring animated decorative elements and a playful space-themed design.  
The design includes floating objects and smooth animations that make the error page more engaging for users.

## Technologies Used
- HTML5
- CSS3
- CSS Animations
- Flexbox

## Features
- Custom 404 error page design
- Animated moving elements around the page
- Space-themed decorative graphics
- Smooth CSS animations
- Modern UI design
- Clean layout and centered content

## Purpose of the Project
This project was built as part of my front-end development practice to improve my understanding of CSS animations, layout positioning, and designing creative error pages.
